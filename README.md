### Cloning:
```
git clone https://github.com/4w4k3/KnockMail.git
```

### Running:
```
cd KnockMail
```

```
sudo su
```

```
pip install -r requeriments.txt
```

```
python knock.py
```

If you have another version of Python:

```
python2.7 knock.py
```
